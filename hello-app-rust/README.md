# hello-app-rust

This is an example of demonstrating how to use a tree-sitter parser in rust.

1. Build the parser in /tree-sitter-hello by running `/tree-sitter-hello/bin/build-nodejs`
2. From here (/hello-app-rust):
  1. `cargo run`
